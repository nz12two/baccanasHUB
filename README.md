# Baccanas Studio™

Landing Page moderna para divulgação de serviços de desenvolvimento Minecraft, criada pela **Baccanas**.

## 📖 Sobre

Este projeto foi desenvolvido pela **equipe da Baccanas** como uma landing page simples, rápida e totalmente estática para apresentação de serviços, portfólio e formas de contato.

## Estrutura

Nosso objetivo é oferecer uma base organizada e fácil de personalizar para quem deseja criar sua própria página.

## ❤️ Inspiração é bem-vinda

Você pode utilizar este projeto como inspiração para criar sua própria landing page.

Não temos nenhum problema em você estudar o código, adaptar ideias ou utilizar partes da estrutura como referência.

Apenas pedimos que **não copie o projeto integralmente nem tente se passar pela Baccanas ou revendê-lo como se fosse de sua autoria**.

## 🚀 Como usar

1. Edite o arquivo `config.js` com as informações do seu projeto:

   * Serviços
   * Portfólio
   * Convite do Discord
   * Webhook
   * Redes sociais

2. Hospede em qualquer serviço de hospedagem estática:

   * GitHub Pages
   * Netlify
   * Vercel
   * Cloudflare Pages
   * ou qualquer outro servidor HTTP.

## 💬 Widget do Discord

Para exibir a quantidade de membros online:

1. Abra as configurações do servidor no Discord.
2. Acesse **Widget**.
3. Ative o Widget do Servidor.
4. Copie o **Server ID**.
5. Cole o ID em `discordGuildId` dentro de `config.js`.

## 📁 Estrutura

```text
index.html   → Estrutura da página
style.css    → Estilos
script.js    → Lógica e animações
config.js    → Configurações editáveis
assets/      → Imagens e arquivos estáticos
```
